This python program map the Keys value to print with with TTL given. Keys, value only prints when TTL is active . After expiry it is not printed .

Folder has file 
1) mapdict_ttl as entire python code .
2) test_mykeyvaluemap.py has  for test cases  for test and validation in pytest framework.
3) mapttl.py is code to use when running tests with Pytest 
4) Pytestrunlog.log is pytest execution log.
